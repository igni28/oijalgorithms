# oijalgorithms
oijalgorithms is a python package written in C containing algorithms often used in the Junior Polish Olympiad in Informatics as well as its High School counterpart, made for learning by writing them in python or using them as a shortcut instead of implementing them yourself.

## Installation
The package requires Python 3 and a C compiler
# To download it from github:
git clone github.com/igni28/oijalgorithms
cd oijalgorithms
python -m pip install -e .
# Then check if it works (in python) for example by running:
import oijalgorithms
print(oijalgorithms.gcd(20, 15) <- this should print 5
