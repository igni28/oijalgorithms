# Changelog

All notable changes to this project will be documented in this file.

## [1.0]
## The first version of the package, it includes:

Basic number theory algorithms:
  - greatest common divisor (`gcd`)
  - least common multiple (`lcm`)
  - fast modular exponentiation (`mod_pow`)
  - the Sieve of Eratosthenes, prime number generator (`sieve`)

Array algorithms:
  - (`binary search`)
  - prefix sums (`prefix_sum`)
  - longest increasing subsequence length (`lis_length`)

graph algorithms:
  - breadth-first search (`bfs`)
  - depth-first search (`dfs`)
  - topological sort (`toposort`)

string searching:
  - Knuth-Morris-Pratt pattern matching algorithm (`kmp_search`)

the `UnionFind` data structure.

## [2.0]
- Added `test_your_function`, a function for comparing the user's Python function with a function from the package on randomly generated tests in order to learn coding yourself. The function can't generate graph tests yet.
